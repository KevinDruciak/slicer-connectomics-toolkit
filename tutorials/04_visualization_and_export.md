# Visualization and Export

Documentation for rendering segmentation results and producing output artifacts.

## Segmentation Overlays

After running NeuronSegmenter, segmentation overlays appear in the slice views as colored regions on top of the EM volume. Overlay opacity is adjustable via the slice view controls. Segment colors can be changed in the **Segmentations** module.

## 3D Rendering

Segmented neurons are automatically rendered as 3D surfaces when the segmentation node's visibility is toggled on in the **Data** module. Surface smoothing is configurable under **Segmentations > Representations > Closed surface > Smoothing factor**.

For combined visualization, I render the EM volume as a semi-transparent volume rendering with segmentation surfaces overlaid, using **Crop Volume** to focus on a region of interest.

## Mesh Export

### From the GUI

Right-click the segmentation in the **Data** module > **Export visible segments to models** > right-click each model > **Export to file** (STL or OBJ).

### From the CLI

```bash
Slicer --python-script scripts/export_neuron_meshes.py \
    --segmentation ./output/labels/sample_labels.nrrd \
    --output-dir ./output/meshes --format stl
```

Exported meshes can be opened in MeshLab, Blender, or ParaView for further rendering.

## Screenshots

High-resolution screenshots are captured via **View > Screenshot** with a 2x or 3x scale factor, or programmatically:

```python
import ScreenCapture
cap = ScreenCapture.ScreenCaptureLogic()
cap.captureImageFromView(
    slicer.app.layoutManager().threeDWidget(0).threeDView(),
    "/path/to/output/3d_view.png"
)
```

## Figure Layout

A typical output figure from this toolkit includes:

1. Raw EM slice with scale bar
2. Same slice with segmentation overlay
3. 3D rendering of segmented neurons
4. Quantitative metrics (generated from the morphology analysis notebook)

## Animation

Rotation animations for presentations are generated by capturing frames during incremental yaw rotations and combining them with ffmpeg:

```bash
ffmpeg -framerate 24 -i /tmp/rotation_%03d.png -c:v libx264 -pix_fmt yuv420p rotation.mp4
```
