## Public Datasets

Download one of these datasets to test the full pipeline on real EM data.

### CREMI Challenge

- **Description:** Three 1250 x 1250 x 125 voxel volumes of serial section transmission EM from *Drosophila melanogaster* brain, with ground truth neuron segmentations and synaptic annotations.
- **Voxel size:** 4 x 4 x 40 nm
- **Link:** [https://cremi.org/](https://cremi.org/)
- **Download:** [https://cremi.org/data/](https://cremi.org/data/)
- **Format:** HDF5 (convert to NRRD using the provided notebooks or `h5totiff`)

### SNEMI3D

- **Description:** 3D segmentation of neurites in serial section EM images from mouse cortex. 1024 x 1024 x 100 voxels.
- **Voxel size:** 6 x 6 x 30 nm
- **Link:** [https://snemi3d.grand-challenge.org/](https://snemi3d.grand-challenge.org/)
- **Format:** TIFF stack

### ISBI 2012 EM Segmentation Challenge

- **Description:** 30-slice serial section TEM stack of *Drosophila* ventral nerve cord with binary membrane labels.
- **Voxel size:** 4 x 4 x 50 nm
- **Link:** [https://imagej.net/events/isbi-2012-segmentation-challenge](https://imagej.net/events/isbi-2012-segmentation-challenge)
- **Format:** TIFF

### FIB-25 (Janelia)

- **Description:** Focused ion beam scanning EM (FIB-SEM) volume of *Drosophila* brain at 8 nm isotropic resolution. Massive dataset suitable for large-scale connectomics.
- **Link:** [https://www.janelia.org/project-team/flyem/data-and-software-release](https://www.janelia.org/project-team/flyem/data-and-software-release)

## Loading Data in 3D Slicer

### NRRD / NIfTI files

1. Open 3D Slicer
2. **File > Add Data**
3. Select the file and click **OK**

### TIFF stacks

1. **File > Add Data**
2. Select all TIFF files in the stack
3. Check **Show Options** and ensure they are loaded as a single volume (not individual images)

### HDF5 files (CREMI)

HDF5 is not natively supported by Slicer. Convert first:

```python
import h5py
import nrrd
import numpy as np

with h5py.File("sample_A_20160501.hdf", "r") as f:
    volume = np.array(f["volumes/raw"])

nrrd.write("sample_A_raw.nrrd", volume, header={"space directions": [[4, 0, 0], [0, 4, 0], [0, 0, 40]]})
```

Then load the resulting `.nrrd` file in Slicer.
