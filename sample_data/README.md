# Sample Data

Datasets used with this toolkit and links to public EM segmentation benchmarks.

## Included

| File | Description | Size |
|---|---|---|
| `mini_em_stack.nrrd` | Synthetic EM-like volume for quick testing (auto-generated by notebooks if absent) | < 1 MB |

## Public Datasets

### ISBI 2012 EM Segmentation Challenge (recommended for quick results)

- 30-slice serial section TEM of *Drosophila* ventral nerve cord with binary membrane labels
- Voxel size: 4 x 4 x 50 nm
- Format: TIFF stack (train-volume.tif, train-labels.tif, test-volume.tif)
- Link: [https://imagej.net/events/isbi-2012-segmentation-challenge](https://imagej.net/events/isbi-2012-segmentation-challenge)

### CREMI Challenge

- Three 1250 x 1250 x 125 voxel volumes of serial section TEM from *Drosophila melanogaster* brain
- Includes ground truth neuron segmentations and synaptic annotations
- Voxel size: 4 x 4 x 40 nm
- Format: HDF5 (requires conversion to NRRD, see below)
- Download: [https://cremi.org/data/](https://cremi.org/data/)

### SNEMI3D

- 1024 x 1024 x 100 voxel EM of mouse cortex neurites
- Voxel size: 6 x 6 x 30 nm
- Format: TIFF stack
- Link: [https://snemi3d.grand-challenge.org/](https://snemi3d.grand-challenge.org/)

### FIB-25 (Janelia)

- FIB-SEM volume of *Drosophila* brain at 8 nm isotropic resolution
- Link: [https://www.janelia.org/project-team/flyem/data-and-software-release](https://www.janelia.org/project-team/flyem/data-and-software-release)

## HDF5 to NRRD Conversion (for CREMI)

```python
import h5py
import nrrd
import numpy as np

with h5py.File("sample_A_20160501.hdf", "r") as f:
    volume = np.array(f["volumes/raw"])

nrrd.write("sample_A_raw.nrrd", volume, header={
    "space directions": [[4, 0, 0], [0, 4, 0], [0, 0, 40]]
})
```
